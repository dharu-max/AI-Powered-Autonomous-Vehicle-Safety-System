# AI-Powered Autonomous Vehicle Safety System

## Overview
This project combines real-time object detection and AI-driven risk analysis to improve road safety and driving assistance.

## Features
- Real-time object detection using YOLO
- Vehicle and pedestrian recognition
- Hazard detection and risk analysis
- Safety alert generation
- Autonomous driving assistance

## Technologies Used
- Python
- YOLOv8
- OpenCV
- NumPy
- Machine Learning

## Applications
- Autonomous Vehicles
- Advanced Driver Assistance Systems (ADAS)
- Smart Transportation
